# Legislatívny asistent pre účtovníkov (Slovensko)
Tento projekt je AI-asistovaný rozhodovací nástroj určený pre účtovníkov a daňových poradcov na Slovensku, ktorí denne pracujú s rýchlo sa meniacou legislatívou.  Cieľom projektu je zjednodušiť orientáciu v slovenskej daňovej a účtovnej legislatíve tým, že poskytuje odpovede výhradne na základe oficiálnych právnych zdrojov.
